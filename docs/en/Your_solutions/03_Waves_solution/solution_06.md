# Wave Equation: Extracting Wave Properties

## Problem Statement
A wave is described by the equation $y(x,t) = 0.05 \sin(2\pi x - 50\pi t)$, where $x$ and $y$ are in meters and $t$ is in seconds. Determine the wave's:
a) Amplitude $A$
b) Wavelength $\lambda$
c) Frequency $f$
d) Wave speed $v$

---

## Step-by-Step Solution

The general wave equation template is:
$$y(x,t) = A \sin(kx - \omega t)$$

By matching the given equation $y(x,t) = 0.05 \sin(2\pi x - 50\pi t)$ to the template, we can directly identify the wave variables:
* **Amplitude ($A$):** $0.05 \text{ m}$
* **Wave Number ($k$):** $2\pi \text{ rad/m}$
* **Angular Frequency ($\omega$):** $50\pi \text{ rad/s}$

### a) Amplitude ($A$)
Read directly from the equation matching:
**$A = 0.05 \text{ meters}$**

### b) Wavelength ($\lambda$)
Use the wave number formula: $k = \frac{2\pi}{\lambda}$
$$2\pi = \frac{2\pi}{\lambda}$$
$$\lambda = \frac{2\pi}{2\pi}$$
**$\lambda = 1 \text{ meter}$**

### c) Frequency ($f$)
Use the angular frequency formula: $\omega = 2\pi f$
$$50\pi = 2\pi f$$
Divide both sides by $2\pi$:
$$f = \frac{50\pi}{2\pi}$$
**$f = 25 \text{ Hz}$**

### d) Wave Speed ($v$)
Use the fundamental wave speed formula: $v = f \cdot \lambda$
$$v = 25 \cdot 1$$
**$v = 25 \text{ m/s}$**
