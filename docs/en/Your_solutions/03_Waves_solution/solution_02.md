# String Harmonics: Speed of a Wave on a Guitar String

## Problem Statement
A guitar string is 64 cm long and has a fundamental frequency (one antinode) of 330 Hz. What is the speed of the wave on this string?

---

## Step-by-Step Solution

To find the speed of the wave, we use the fundamental wave equation:
**v = f × λ** (Speed = Frequency × Wavelength)

The problem gives us the length of the string (L) and the frequency (f). We must first determine the wavelength (λ) from the string's length.

### Step 1: Find the Wavelength (λ)
The string is vibrating at its **fundamental frequency** with "one antinode". This means the vibrating string forms exactly half of a full wave. Therefore, the full wavelength is twice the length of the string.
* Formula: **λ = 2L**

First, convert the string length to standard SI units (meters):
* L = 64 cm = 0.64 m

Now, calculate the wavelength:
* λ = 2 × 0.64 m
* λ = 1.28 m

### Step 2: Calculate the Wave Speed (v)
Now that we have both the frequency (f) and the wavelength (λ), we can plug them into the main wave equation.
* Frequency (f) = 330 Hz
* Wavelength (λ) = 1.28 m

* v = f × λ
* v = 330 × 1.28
* v = 422.4 m/s

**Final Answer:**
The speed of the wave on the guitar string is **422.4 m/s**.
